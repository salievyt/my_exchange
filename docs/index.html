1. Введение

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей мобильного приложения «My Exchange» («Моя волюта») (далее — «Приложение»), разработанного для управления обменными валютными операциями, кассовым учётом и аналитикой.

Используя Приложение, вы соглашаетесь с условиями настоящей Политики. Если вы не согласны с условиями, пожалуйста, не используйте Приложение.

2. Какую информацию мы собираем

2.1. Данные, предоставляемые пользователем

  • Учётные данные: имя пользователя (username) и пароль для авторизации в системе
  • Персональная информация: имя, фамилия, номер телефона, email (заполняются администратором при создании учётной записи)
  • Данные операций: информация о совершённых обменных операциях, включая сумму, валюту, курс, имя клиента (указывается по желанию)
  • Данные кассовых операций: информация об открытии/закрытии смен, остатках валют, внесениях и выдачах

2.2. Данные, собираемые автоматически

  • IP-адрес: для ведения журнала аудита действий пользователей
  • Информация об устройстве: тип устройства, версия ОС (только для обеспечения совместимости)
  • Дата и время: отметки времени для всех действий в системе

2.3. Данные, которые НЕ собираются

  • Геолокация (GPS)
  • Контакты и адресная книга
  • Фотографии и медиафайлы
  • Данные биометрии
  • История браузера
  • Персональные данные клиентов (Приложение не хранит данные клиентов обменных пунктов централизованно)

3. Цели сбора данных

Собранные данные используются исключительно для:

  • Авторизации и аутентификации пользователей в системе
  • Выполнения обменных операций (покупка/продажа валюты)
  • Ведения кассового учёта (открытие/закрытие смен, учёт остатков)
  • Формирования аналитики и отчётов для управленческих целей
  • Аудита безопасности — фиксация всех действий пользователей для предотвращения мошенничества
  • Технической поддержки и улучшения работы Приложения

4. Правовое основание обработки

Обработка персональных данных осуществляется на следующих основаниях:

  • Исполнение договора между пользователем и администратором системы (использование Приложения для выполнения служебных обязанностей)
  • Законный интерес — обеспечение безопасности и целостности финансовых операций
  • Согласие пользователя на обработку данных (принимается при первом входе в систему)

5. Передача данных третьим лицам

Мы НЕ передаём персональные данные пользователей третьим лицам, за исключением следующих случаев:

  • Хостинг-провайдер: данные хранятся на серверах хостинг-провайдера, обеспечивающего работу API
  • Правоохранительные органы: по законному требованию в соответствии с законодательством Кыргызской Республики

Приложение не использует сторонние SDK для аналитики, рекламы или отслеживания пользователей.

Используемые технологии

  • API: Django REST Framework (собственный сервер)
  • База данных: PostgreSQL (собственный сервер)
  • Аутентификация: JWT (JSON Web Tokens)
  • Хостинг: выделенный сервер, Кыргызстан

6. Хранение и защита данных

6.1. Меры безопасности

  • Все данные передаются по зашифрованному протоколу HTTPS
  • Пароли хранятся в захэшированном виде (bcrypt)
  • JWT-токены имеют ограниченный срок действия (8 часов для access, 7 дней для refresh)
  • Весь доступ к API требует аутентификации
  • Ведётся полный аудит всех действий пользователей
  • Применяется ролевая модель доступа (кассир, старший кассир, администратор)

6.2. Срок хранения

Данные пользователей и операций хранятся в течение всего срока эксплуатации Приложения. После удаления учётной записи данные обезличиваются и могут храниться в обезличенном виде для формирования статистики.

7. Права пользователей

Вы имеете право:

  • Доступ к данным: запросить информацию о том, какие данные о вас хранятся
  • Исправление данных: потребовать исправления неверных или устаревших данных
  • Удаление данных: запросить удаление вашей учётной записи и связанных с ней данных
  • Ограничение обработки: потребовать ограничения обработки ваших данных
  • Экспорт данных: получить копию ваших данных в машиночитаемом формате

Для реализации ваших прав обратитесь к администратору вашей организации или напишите на saliev@gmail.com.

Запрос на удаление учётной записи:
Для удаления учётной записи обратитесь к администратору системы. Администратор может деактивировать учётную запись и удалить связанные с ней персональные данные в течение 30 дней с момента запроса.

8. Файлы cookie и локальное хранение

Приложение использует локальное хранилище устройства (Flutter Secure Storage) для хранения:

  • JWT-токенов для автоматической аутентификации
  • Настроек темы оформления (светлая/тёмная)
  • Языковых предпочтений

Эти данные хранятся только на вашем устройстве и не передаются третьим лицам.
